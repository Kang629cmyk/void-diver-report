# VOID DIVER 資料整理

這個資料夾可以直接上傳到 GitHub，並用 GitHub Pages 分享。

入口：

- `index.html`
- `drop_pool_report/VOID_DIVER_drop_pools_report.html`
- `VOID_DIVER_最終配裝強度圖表.html`

GitHub Pages 設定：

1. 建立一個新 repository。
2. 把本資料夾內容上傳到 repository 根目錄。
3. 到 Settings > Pages。
4. Source 選 Deploy from a branch。
5. Branch 選 main，資料夾選 /root。
6. 儲存後等待 GitHub 產生網址。
