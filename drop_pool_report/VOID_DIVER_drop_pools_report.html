VOID DIVER 掉落資料總表

依據 遊戲版本: 6000.3.5f2 (Steam build 23857102)。這裡只保留最常用的查詢入口，舊的分散表格已整合進「掉落來源整合查詢」。

建議優先使用「掉落來源整合查詢」：查物品會列出地圖區域與來源，查怪物會列出它會掉什麼，查箱子會列出箱子內容。
掉落來源整合查詢用物品名、怪物名、箱子名或 ID 直接搜尋，反查地圖區域、怪物、箱子與掉落權重。CSV：08_unified_loot_sources.csv 地圖箱子搜括掉落專門查看各地圖區域箱子群組、箱子掉落池與物品明細。CSV：items / summary / difficulty 怪物弱點與基礎數值查看怪物中文名、弱點屬性、基礎 HP / 攻擊 / 防禦，以及不同難度倍率後數值。CSV：06_monster_weaknesses.csv 難度掉落倍率查看各難度對箱子生成、掉落生成、稀有度抽取的倍率。CSV：00_difficulty_drop_modifiers.csv 掉落池難度試算保留原本依難度試算掉落池稀有度權重的頁面。CSV：01_drop_pools_by_difficulty.csv 最終配裝強度圖表角色傷害公式、配裝面板、套裝差異與圖表總覽。
